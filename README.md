# pxt-rgb-led

- for RGB LEDs
- Support components with pins for R, G, B respectively and a common anode

![](./rgb_led_board.jpg)
![](./rgb_led.jpg)

## Blocks

### Create

Create an RGB LED instance.

Caller provides the pins for R, G, B respectively.

### Show Color

Show color of RGB tuple, each of range 0-255.
